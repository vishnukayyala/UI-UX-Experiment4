# UI-UX-Experiment4

## Aim:
To design a wireframe diagram for a generic application (Shopright) using Balsamiq.

## Algorithm:
1. Open Balsamiq Wireframes and create a new project.

2. Design the Welcome/Splash Screen with the app name Shopright and a Get Started button.

3. Create the Login/Signup Screen allowing users to proceed to the application.

4. Design the Home/Dashboard Screen with a search bar and product cards.

5. Create the Content/Product Listing Screen displaying multiple products with Add to Cart options.

6. Design the Checkout/Final Action Screen with user details and Confirm Order button.

7. Arrange all screens properly and export the wireframe diagram.
